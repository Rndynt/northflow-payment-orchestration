# Merchant Backend Example

This folder contains backend-only integration examples. Do not copy Northflow API keys, signing secrets, service tokens, provider credentials, database URLs, or webhook secrets into frontend/public environment variables.

Files:

- `.env.example`: safe backend environment variable names.
- `sdk-checkout-flow.ts`: TypeScript SDK checkout flow for a merchant backend.
- `rest-checkout-flow.md`: cURL flow for a merchant backend.
